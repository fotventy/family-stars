# Сборка проекта

## Проверка кода

- **Линт:** `npm run lint` (перед первым запуском выполните `npm install`)
- **Разработка:** `npm run dev` — приложение собирается и работает (в т.ч. страница логина)
- **Production build:** `npm run build` — известна ошибка парсера SWC в `src/app/login/page.tsx` («Expected jsx identifier»). В dev-режиме та же страница собирается без ошибок.

## Исправления, уже внесённые в код

- В `src/app/page.tsx` и `src/app/login/page.tsx` исправлен пропущенный `>` в теге span: `}}|</span>` → `}}>|</span>`.
- Стили страницы логина вынесены в `src/app/login/login.css` (вместо styled-jsx в компоненте).
- Next.js можно обновить до актуального 14.x/15.x с патчем безопасности после проверки совместимости.

## Установка зависимостей

Next.js и остальные зависимости ставятся через npm в каталоге проекта:

```bash
npm install
```

Команда `brew install next` в системе не нужна — Next используется как зависимость проекта.
